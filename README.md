# neuro tranning

이 Vault는 다음 두 커스텀 GPT가 함께 사용하는 지식 저장소입니다.

- 뉴로트레이닝 스터디 GPT: 연구, 학습, 질문, 실험 기록
- 독학골프가이드 GPT: 골프 학습 적용과 유튜브 아이디어·구성안·대본 제작

## 일상 사용

1. 정리되지 않은 자료는 `00 Inbox/`에 넣습니다.
2. 공부 중인 내용은 `20 Study`에서 발전시킵니다.
3. 근거가 확인된 지식은 `10 Shared Knowledge`로 옮깁니다.
4. 유튜브 결과물은 `30 YouTube`에 저장합니다.
5. 새 GPT 대화에서는 `이어서 시작`이라고 입력합니다.
6. 대화를 끝낼 때는 `정리하고 저장`이라고 입력합니다.

## 중요한 제한

GPT의 쓰기 Action은 확인을 요구하거나 모델 판단에 따라 생략될 수 있습니다. Obsidian이 닫혀 있으면 GitHub에서 변경된 노트는 다음 실행과 자동 pull 때 로컬에 반영됩니다.
