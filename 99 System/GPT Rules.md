# GPT 공통 규칙

## 읽기

1. 답변 전에 `99 System/Knowledge Index.md`와 관련 Markdown 노트를 검색합니다.
2. 검색 결과가 질문과 실제로 관련 있는지 확인합니다.
3. Vault에서 읽은 사실과 일반 지식을 구분합니다.
4. 사용한 Vault 노트의 경로를 답변이나 결과물에 기록합니다.

## 쓰기

1. 기존 파일을 갱신하기 전에 최신 파일 내용과 SHA를 읽습니다.
2. SHA가 바뀌었거나 충돌이 의심되면 기존 파일을 덮어쓰지 않습니다.
3. 충돌 시 `YYYY-MM-DD-HHMM-제목-draft.md` 형식의 새 초안을 만듭니다.
4. 사용자가 요청하지 않은 삭제, 이동, 대량 덮어쓰기를 하지 않습니다.
5. 계정 토큰, 비밀번호, 개인정보, 의료 기록을 저장하지 않습니다.

## 대화 기억

1. 새 사용자 메시지를 받으면 직전 질문과 답변을 구조화해 자기 대화 폴더에 저장합니다.
2. 첫 메시지라 직전 대화가 없으면 저장을 생략합니다.
3. `이어서 시작`을 받으면 최근 대화 요약과 관련 지식을 먼저 읽습니다.
4. `정리하고 저장`을 받으면 현재 세션의 마지막 답변까지 요약해 저장합니다.
5. 전체 원문보다 핵심 질문, 답변, 결정, 새 지식, 다음 작업을 우선합니다.

## 파일 이름

- 대화 요약: `YYYY-MM-DD-HHMM-주제.md`
- 스터디 노트: `YYYY-MM-DD-주제.md`
- 유튜브 구성안: `YYYY-MM-DD-제목-outline.md`
- 유튜브 대본: `YYYY-MM-DD-제목-script.md`
