# 출처와 근거 규칙

## 근거 수준

- `high`: 여러 고품질 연구나 권위 있는 지침이 일관되게 지지
- `moderate`: 제한된 수의 연구가 대체로 지지하지만 불확실성이 존재
- `low`: 소규모 연구, 관찰 자료, 전문가 의견 중심
- `hypothesis`: 검증되지 않은 가설, 경험적 추정, 콘텐츠 아이디어

## 기록 원칙

1. 연구 제목, 저자 또는 기관, 발표 연도, URL 또는 식별자를 기록합니다.
2. 연구 결과와 GPT의 해석을 분리합니다.
3. 표본, 대조군, 측정 방법, 연구 한계를 가능한 범위에서 기록합니다.
4. 뉴로트레이닝 효과를 치료나 완치로 단정하지 않습니다.
5. 유튜브 제목과 대본이 원 연구보다 강한 주장을 하지 않게 합니다.
6. 출처를 확인할 수 없는 내용은 `hypothesis`로 표시하고 공용 지식으로 승격하지 않습니다.

## 골프 적용

골프 연습에 뉴로트레이닝이나 운동학습 원리를 적용할 때는 과학적 근거, 실전 적용 아이디어, 개인 경험을 별도 구획으로 작성합니다.
